БЫСТРЫЙ СТАРТ
─────────────

1. Скачайте и установите XAMPP:
   https://www.apachefriends.org/download.html

2. Запустите файл: install.bat
   (двойной клик)

3. В открывшемся phpMyAdmin:
   - Нажмите "Импорт"
   - Выберите файл database.sql
   - Нажмите "Вперёд"

4. Создайте пользователя по умолчанию:
   - Откройте http://localhost/battery_tracker/setup_user.php
   - Или зарегистрируйтесь на странице регистрации

5. Войдите в систему с учетными данными:
   Логин: user123
   Пароль: pas123

6. Готово! Теперь вы можете использовать систему.
