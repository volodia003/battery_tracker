БЫСТРЫЙ СТАРТ
─────────────

1. Скачайте и установите XAMPP:
   https://www.apachefriends.org/download.html

2. Запустите файл: install.bat
   (двойной клик)

3. В открывшемся phpMyAdmin:
   - Нажмите "Импорт"
   - Выберите файл database.sql
   - Нажмите "Вперёд"

4. Готово! Сайт откроется автоматически.


ПОВТОРНЫЙ ЗАПУСК
────────────────

Запустите файл: start.bat


ССЫЛКИ
──────

Сайт:        http://localhost/battery_tracker/
phpMyAdmin:  http://localhost/phpmyadmin/


ФУНКЦИОНАЛ
──────────

✓ Учет устройств (телефоны, ноутбуки, наушники, шуруповерты)
✓ Журнал зарядок с типами (обычная, быстрая, беспроводная)
✓ Автоматическая деградация батареи при зарядке
✓ Графики здоровья батареи
✓ Прогноз срока службы
✓ Уведомления о низком здоровье (<80%)
✓ Фильтрация и поиск


ДЕГРАДАЦИЯ БАТАРЕИ
──────────────────

При каждой зарядке здоровье батареи уменьшается:

  Обычная зарядка:     0.05% за полный цикл
  Быстрая зарядка:     0.08% за полный цикл
  Беспроводная:        0.06% за полный цикл


ТРЕБОВАНИЯ
──────────

- Windows 7/8/10/11
- XAMPP (Apache + MySQL + PHP)
- Браузер (Chrome, Firefox, Edge)
