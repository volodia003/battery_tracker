                  BATTERY TRACKER - ИНСТРУКЦИЯ ПО РАЗВЕРТЫВАНИЮ
════════════════════════════════════════════════════════════════════════════════

ТРЕБОВАНИЯ: Windows 7+, 500 MB свободного места, любой браузер


ШАГ 1: УСТАНОВКА OPEN SERVER
─────────────────────────────

1. Скачайте Open Server с официального сайта:
   https://ospanel.io/download/

2. Запустите установщик Open Server
   [СКРИНШОТ 1: Окно установщика Open Server с кнопкой "Установить"]

3. Выберите папку установки (по умолчанию C:\OpenServer)
   [СКРИНШОТ 2: Выбор папки установки]

4. Дождитесь завершения установки


ШАГ 2: КОПИРОВАНИЕ ФАЙЛОВ ПРОЕКТА
───────────────────────────────────

1. Скопируйте папку проекта в:
   C:\OpenServer\domains\battery_tracker\

2. Структура должна выглядеть так:
   C:\OpenServer\domains\battery_tracker\index.php
   C:\OpenServer\domains\battery_tracker\database.sql
   C:\OpenServer\domains\battery_tracker\config\
   ...остальные файлы

   [СКРИНШОТ 3: Проводник Windows с папкой battery_tracker в domains]


ШАГ 3: ЗАПУСК OPEN SERVER
───────────────────────────

1. Запустите Open Server (иконка на рабочем столе или в меню Пуск)
   [СКРИНШОТ 4: Иконка Open Server на рабочем столе]

2. В трее появится красный флажок - кликните на него
   [СКРИНШОТ 5: Красный флажок Open Server в трее]

3. Выберите "Запустить" → выберите Apache и MySQL
   [СКРИНШОТ 6: Меню Open Server с кнопкой "Запустить"]

4. Флажок станет зелёным - сервер запущен
   [СКРИНШОТ 7: Зелёный флажок Open Server]


ШАГ 4: СОЗДАНИЕ БАЗЫ ДАННЫХ
─────────────────────────────

1. Кликните на зелёный флажок Open Server → "Дополнительно" → "phpMyAdmin"
   [СКРИНШОТ 8: Меню Open Server с пунктом phpMyAdmin]

2. В phpMyAdmin нажмите вкладку "Импорт"
   [СКРИНШОТ 9: Интерфейс phpMyAdmin, вкладка "Импорт"]

3. Нажмите "Выберите файл" → найдите файл database.sql в папке проекта
   [СКРИНШОТ 10: Кнопка выбора файла в phpMyAdmin]

4. Нажмите "Вперёд" внизу страницы
   [СКРИНШОТ 11: Кнопка "Вперёд" в phpMyAdmin]

5. Увидите сообщение "Импорт успешно завершён"
   [СКРИНШОТ 12: Сообщение об успешном импорте]


ШАГ 5: ЗАПУСК ПРИЛОЖЕНИЯ
──────────────────────────

1. Откройте браузер и перейдите по адресу:
   http://battery_tracker/

   ИЛИ

   http://localhost/battery_tracker/

2. Откроется страница входа
   [СКРИНШОТ 13: Страница входа Battery Tracker]

3. Войдите с учетными данными:
   Логин:  user123
   Пароль: pas123

4. Готово! Вы в системе
   [СКРИНШОТ 14: Главная страница системы после входа]


ПОВТОРНЫЙ ЗАПУСК
─────────────────

При следующем включении компьютера:
1. Запустите Open Server (иконка на рабочем столе)
2. Кликните на флажок → "Запустить"
3. Откройте http://battery_tracker/ в браузере


РЕШЕНИЕ ПРОБЛЕМ
────────────────

ПРОБЛЕМА: Порт 80 занят
РЕШЕНИЕ: 
- Кликните на флажок Open Server → "Настройки" → "Сервер"
- Измените порт Apache с 80 на 8080
- Перезапустите сервер
- Теперь адрес: http://localhost:8080/battery_tracker/

ПРОБЛЕМА: Не открывается сайт
РЕШЕНИЕ:
- Убедитесь что флажок зелёный (сервер запущен)
- Проверьте файлы скопированы в C:\OpenServer\domains\battery_tracker\

ПРОБЛЕМА: Ошибка подключения к БД
РЕШЕНИЕ:
- Убедитесь что импортировали database.sql через phpMyAdmin
- Проверьте что база battery_tracker_db создана
