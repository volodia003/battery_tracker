                    BATTERY TRACKER - ИНСТРУКЦИЯ ПО ЭКСПЛУАТАЦИИ
════════════════════════════════════════════════════════════════════════════════

1. ВХОД В СИСТЕМУ
─────────────────

   1.1. Откройте браузер и перейдите по адресу:
        http://localhost/battery_tracker/login.php

   1.2. Введите ваши учетные данные:
        - Имя пользователя
        - Пароль

   1.3. Нажмите кнопку "Войти"

   Примечание: Если у вас нет аккаунта, нажмите ссылку "Зарегистрироваться"


2. ГЛАВНАЯ СТРАНИЦА (ДАШБОРД)
──────────────────────────────

   После входа вы увидите главную страницу с общей статистикой:
   
   - Всего устройств в системе
   - Количество устройств, требующих внимания (здоровье < 80%)
   - Общее количество зарядок
   - Среднее здоровье батарей

   На главной странице также отображаются:
   - Список устройств с низким здоровьем батареи
   - Последние 5 зарядок
   - График здоровья батарей всех устройств


3. УПРАВЛЕНИЕ УСТРОЙСТВАМИ
───────────────────────────

   3.1. ДОБАВЛЕНИЕ УСТРОЙСТВА
        - Перейдите в раздел "Устройства" в меню
        - Нажмите кнопку "Добавить устройство"
        - Заполните форму:
          * Тип устройства (телефон, ноутбук, наушники, шуруповерт, планшет)
          * Модель устройства
          * Емкость батареи (в mAh)
          * Дата покупки
          * Здоровье батареи (по умолчанию 100%)
          * Заметки (необязательно)
        - Нажмите "Сохранить"

   3.2. ПРОСМОТР УСТРОЙСТВА
        - В списке устройств нажмите на название или иконку глаза
        - Вы увидите детальную информацию:
          * Основные характеристики
          * Текущее здоровье батареи
          * График изменения здоровья
          * История зарядок
          * Прогноз срока службы батареи

   3.3. РЕДАКТИРОВАНИЕ УСТРОЙСТВА
        - В списке устройств нажмите иконку карандаша
        - Измените необходимые данные
        - Нажмите "Сохранить"

   3.4. УДАЛЕНИЕ УСТРОЙСТВА
        - В списке устройств нажмите иконку корзины
        - Подтвердите удаление
        - ВНИМАНИЕ: Все связанные записи о зарядках также будут удалены!

   3.5. ФИЛЬТРАЦИЯ И ПОИСК
        - Используйте поле поиска для поиска по модели или заметкам
        - Фильтруйте по типу устройства
        - Фильтруйте по здоровью батареи (хорошее/низкое)


4. ЖУРНАЛ ЗАРЯДОК
──────────────────

   4.1. ДОБАВЛЕНИЕ ЗАПИСИ О ЗАРЯДКЕ
        - Перейдите в раздел "Журнал зарядок"
        - Нажмите кнопку "Добавить зарядку"
        - Заполните форму:
          * Выберите устройство
          * Дата и время зарядки
          * Начальный процент заряда
          * Конечный процент заряда
          * Тип зарядки (обычная, быстрая, беспроводная)
          * Продолжительность в минутах
          * Заметки (необязательно)
        - Нажмите "Сохранить"

   4.2. АВТОМАТИЧЕСКАЯ ДЕГРАДАЦИЯ
        При добавлении зарядки система автоматически уменьшает здоровье батареи:
        - Обычная зарядка: -0.05% за полный цикл (0-100%)
        - Быстрая зарядка: -0.08% за полный цикл
        - Беспроводная: -0.06% за полный цикл

   4.3. ПРОСМОТР СТАТИСТИКИ
        В верхней части страницы отображаются:
        - Всего записей о зарядках
        - Средний процент заряда
        - Среднее время зарядки
        - Количество быстрых зарядок

   4.4. ФИЛЬТРАЦИЯ ЗАРЯДОК
        - По устройству
        - По типу зарядки
        - По диапазону дат

   4.5. РЕДАКТИРОВАНИЕ И УДАЛЕНИЕ
        - Нажмите иконку карандаша для редактирования
        - Нажмите иконку корзины для удаления записи


5. АНАЛИТИКА
─────────────

   Раздел "Аналитика" предоставляет детальную информацию:

   5.1. ОБЩАЯ СТАТИСТИКА
        - Всего устройств
        - Устройства, требующие внимания
        - Всего зарядок
        - Среднее здоровье батарей

   5.2. ГРАФИК ЗДОРОВЬЯ БАТАРЕИ
        - Выберите устройство из выпадающего списка
        - График покажет изменение здоровья батареи во времени

   5.3. ПРОГНОЗ СРОКА СЛУЖБЫ
        Система автоматически рассчитывает:
        - Примерное оставшееся время работы батареи
        - Потерю здоровья в месяц
        - Количество дней до достижения 20% здоровья
        - Рекомендации по замене

   5.4. ГРАФИК ЗАРЯДОК ПО МЕСЯЦАМ
        - Показывает количество зарядок каждого типа
        - Данные группируются по месяцам

   5.5. РАСПРЕДЕЛЕНИЕ ПО ТИПАМ УСТРОЙСТВ
        - Круговая диаграмма показывает соотношение типов устройств

   5.6. ТОП УСТРОЙСТВ
        - Рейтинг устройств по количеству зарядок
        - Отображается здоровье каждого устройства


6. НАСТРОЙКИ ТЕМЫ
──────────────────

   6.1. ПЕРЕКЛЮЧЕНИЕ ТЕМЫ
        - Нажмите иконку луны/солнца в верхнем правом углу
        - Доступны темы: светлая и темная
        - Настройка сохраняется для вашего аккаунта


7. ВЫХОД ИЗ СИСТЕМЫ
────────────────────

   - Нажмите кнопку "Выход" в меню
   - Вы будете перенаправлены на страницу входа


8. РЕКОМЕНДАЦИИ ПО ИСПОЛЬЗОВАНИЮ
──────────────────────────────────

   8.1. ДЛЯ ТОЧНОЙ СТАТИСТИКИ
        - Регулярно записывайте зарядки устройств
        - Обновляйте информацию о здоровье батареи
        - Используйте правильный тип зарядки

   8.2. ОБСЛУЖИВАНИЕ БАТАРЕЙ
        - Обращайте внимание на уведомления о низком здоровье (<80%)
        - Избегайте частых быстрых зарядок
        - Рассмотрите замену батареи при здоровье < 50%

   8.3. ОПТИМИЗАЦИЯ РАБОТЫ
        - Используйте фильтры для быстрого поиска
        - Просматривайте аналитику для понимания паттернов использования
        - Добавляйте заметки для важной информации


9. ЧАСТО ЗАДАВАЕМЫЕ ВОПРОСЫ
────────────────────────────

   Q: Как рассчитывается деградация батареи?
   A: Деградация зависит от типа зарядки и процента заряженного. Формула:
      Потеря = (конечный % - начальный %) / 100 * коэффициент типа зарядки

   Q: Можно ли восстановить удаленное устройство?
   A: Нет, удаление необратимо. Будьте осторожны при удалении.

   Q: Почему прогноз не отображается?
   A: Для прогноза требуется минимум 2 записи об изменении здоровья батареи.

   Q: Как добавить новый тип устройства?
   A: Используйте опцию "Другое" при добавлении устройства.


10. БЕЗОПАСНОСТЬ
─────────────────

   - Регулярно меняйте пароль
   - Не делитесь учетными данными
   - Выходите из системы после использования на общих компьютерах
